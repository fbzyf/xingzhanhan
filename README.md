# 上海星展瀚电子 企业官网

## 项目简介

这是 **上海星展瀚电子信息科技有限公司**（Shanghai StarVast Electronic Information Technology Co., Ltd）的企业展示型官方网站。

网站用于向客户展示公司的基本信息、代理品牌和产品应用领域，便于客户了解公司业务并取得联系。

## 网站地址

直接双击打开 `index.html` 即可在浏览器中预览网站。

## 项目结构

```
xingzhanhan/
│
├── index.html          # 主页面（单页滚动式，包含所有内容区域）
├── logo.png            # 公司 Logo 图片
├── README.md           # 项目说明文档（本文件）
│
├── css/
│   └── style.css       # 自定义样式（渐变、动画、悬停效果等）
│
└── js/
    └── main.js         # 交互逻辑（导航栏、滚动动画、移动端菜单等）
```

## 技术方案

- **HTML5** - 页面结构
- **Tailwind CSS**（CDN引入） - 响应式布局和样式
- **原生 JavaScript** - 交互逻辑
- **Noto Sans SC**（Google Fonts） - 中文字体

**特点：**
- 纯静态网站，无需后端/数据库
- 无需安装任何依赖，直接打开即可运行
- 响应式设计，手机/平板/电脑均可正常浏览
- 所有图标使用内联 SVG，无需额外图片文件

## 如何部署

### 方式一：本地预览
直接在浏览器中打开 `index.html` 文件即可。

### 方式二：部署到服务器
将整个项目文件夹上传到任意 Web 服务器（如 Nginx、Apache）的网站根目录即可。

### 方式三：静态托管平台
可部署到以下免费平台：
- **GitHub Pages** - 将代码推送到 GitHub 仓库，开启 Pages 功能
- **Vercel** - 导入仓库即可自动部署
- **Netlify** - 拖拽文件夹即可完成部署

## 维护指南

如需修改网站内容：

1. **修改文字内容** - 编辑 `index.html` 中对应区域的文字即可
2. **修改颜色** - 编辑 `index.html` 顶部 `<script>` 标签中的 `tailwind.config` 配置
3. **修改动画效果** - 编辑 `css/style.css` 中对应的样式
4. **修改交互逻辑** - 编辑 `js/main.js`
5. **更换 Logo** - 用新的 Logo 文件替换根目录下的 `logo.png`

## 已知注意事项

- Tailwind CSS 和 Google Fonts 通过 CDN 引入，首次加载需要网络连接
- 如果在国内网络环境中 Google Fonts 加载较慢，可以考虑替换为国内 CDN 镜像
- Logo 文件建议使用正方形 PNG 格式，推荐尺寸 200x200 像素以上
